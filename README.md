# 📚 Java学習リポジトリ

[一週間で身につくJava言語の基本](https://java.sevendays-study.com/index.html)を参考に、Javaの基礎を学習するための個人リポジトリです。

## 🎯 学習の目的

- Javaプログラミングの基礎を習得する
- オブジェクト指向プログラミングの概念を理解する
- 毎日コツコツと学習を進める習慣をつける
